# 10 Branch Styles & Sub Mind Map Collapse

> 🧩**MarginNote 4 supports many styles of mind map branches. Choose a branch style based on the structure of your content:**
> 
> - **Hierarchical**: topic → subtopic → key point
>   - **`Tree 0/1/2/3/4`**
>     - Subject knowledge trees; topics and subtopics in literature reviews
> - **Process / timeline**: Step 1 → 2 → 3, or today → tomorrow
>   - **`Line 0/1/2`**
>     - Study-plan timelines, project milestones, algorithm/operation step sequences, experiment process logs
> - **Comparison**: Option A vs B, pros vs cons
>   - **`Two Way`**
>     - Requirements ⇄ implementation, goals ⇄ metrics, and other paired correspondences
> - **Bundle into a group**: put cards from the same module together
>   - **`Framework`**
>     - Gather a set of nodes together to highlight them as a single "framework" unit

## 1 How to Switch Branch Styles

> 💡**Global setting**: After opening the `mind map` view, tap More - `mind map` - `Branch Style`. The change affects all nodes that have not been manually modified‼️.
> 
> **Single-card setting**: card `pop-up menu` - `basic` - `...(More)`. The change affects only that single card.

> 💡The `Branch Style` set in the mind map settings panel changes **all nodes in the mind map that have not been manually modified**. If you only want to modify a **specific card** and its child-card branches, you can do so via the card `pop-up menu` → `basic` → `Branch`.

## 3 Lossless Conversion Between Branches and Sub Mind Maps

MarginNote 4 supports not only lossless conversion between branch styles, but also lossless conversion between branches and sub mind maps.

> 💡When a card node holds too much information, you can `collapse the branch into a sub mind map` to make the information easier to organize and read.

Select a card and tap `pop-up menu - advanced` - `Collapse into Sub Mind Map`. A "sub mind map card" (made up of that card and its child nodes) is generated in the original position.
