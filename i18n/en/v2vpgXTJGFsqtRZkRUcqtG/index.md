# Translating and Saving Translations

> 💡This page covers four ways to obtain and save translations in MarginNote: instant in-document translation, AI translation and AI OCR, the Research sidebar (including Custom URL), and plugin translation. You can insert a translation as an extend note, copy it to the clipboard, or save it as a card for side-by-side comparison and later organization.
> 
> Recommendations:
> 
> - Split-screen side panel: use the built-in Research → Translate sidebar to translate long passages; the sidebar refreshes automatically.
> - Fastest to get started: use Built-in Translation (Max feature) → insert as an extend note (good for comparing while reading)
> - High quality / automation: use a plugin such as DeepL → Translate or AI translation, combined with AI OCR for recognition → translation → organization, to improve reliability for long sentences and natural phrasing.
> - Authoritative terminology: use built-in Research → Dictionary Search, or connect authoritative sites via built-in Research → Custom URL, to ensure precise definitions and example sentences.

## 1 Built-in Translation (Max feature)

> 💡The most beginner-friendly way to translate in text and extend note scenarios.

### 1.1 Translate

Hand tool - Document

Use the `hand tool` to select text (as shown above), then tap `Translate` in the `pop-up menu` to translate the selected text into the target language.

![More - Translate More - Translate ](image/91d01468f07acc09.webp)

### 1.2 Changing the Target Language

Tap `Language` to change the target language

> 💡If a translation is wrong, check the target language first.

![Tap Language to change the target language Tap Language to change the target language ](image/6791ded5fc7a54e4.webp)

Built-in Translation supports 13 target languages: English, Simplified Chinese, Traditional Chinese, Cantonese, Classical Chinese, Portuguese, French, German, Italian, Spanish, Japanese, Korean, and Russian.

> 💡Use built-in **Translation frequently?**
> 
> We recommend checking `Translate` under `pop-up menu` - `More` - `Customize` to add `Translate` to the first-level menu for one-tap access.
> 
> ![Add Translate to the first-level menu Add Translate to the first-level menu ](image/eafe97f009b6dcb7.webp)

### 1.3 Saving Translations

A translation result can be created directly as an extend note or copied.

#### 1.3.1 Adding a Translation to an Extend Note

After the translation is generated, tap `extend note`, and the translation is inserted into the document as an `extend note`, side by side with the original.

![After a successful translation, tap Extend note After a successful translation, tap Extend note ](image/ca151ec050fc2613.webp)

![Saved as an Extend note after the translated paragraph; the Extend note can be Collapsed Saved as an Extend note after the translated paragraph; the Extend note can be Collapsed ](image/fe5e5d2899f8311f.webp)

> 💡Adding it directly as an `extend note` makes side-by-side comparison with the original easy. An `extend note` can be `collapsed` and expanded as needed while reading, which helps with review and annotation. For more extend note operations, see: [Extend Note ① | Basics: Freely Open Up Note Space Between the Lines](../vYi6Yu4oCudNCr2zDuNQkj/index.md).
> 
> ![Open a collapsed Extend note Open a collapsed Extend note ](image/553ad5947cac201b.webp)

#### 1.3.2 Copying Translated Text to the Clipboard

Tap `Copy`, and the translation is automatically copied to the clipboard; pasting it directly onto a blank area of the document saves it as a `Text box`.

![After a successful translation, tap Copy and the translation is automatically copied to the system clipboard After a successful translation, tap Copy and the translation is automatically copied to the system clipboard ](image/ca151ec050fc2613.webp)

> 💡`Copy` is used for pasting and organizing across pages or across apps.

## 2 AI Translation

### 2.1 Translating in an AI Chat

Use MarginNote 4's built-in AI feature to translate (requires version 4.2.0 or later); it is well suited to optimizing long sentences and natural expression. The generated result can likewise be copied or saved as a card.

![](image/5b9992488f6779ce.webp)

![The translation can be copied or saved as a card The translation can be copied or saved as a card ](image/2668f5cb4a704355.webp)

### 2.2 Translating with AI OCR

In the document interface, do the following:

- Double-click the **hand tool** or **any excerpt tool**;
- In the settings panel that pops up, check the `AI OCR` feature; afterward, when making cards from excerpts, the AI will automatically recognize the excerpt content and convert it into editable text;
- In the `AI Card Creation Template (Max feature)`, enter a prompt or choose a preset prompt to have the AI automatically organize, translate, summarize, create flashcards, and more after recognizing the content.

## 3 Research → Translate

### 3.1 Using Built-in Research → Translate

Research

1. In a study set, tap `Research` under More (as shown by the icon above) to open the Research browser.

   ![Select Translate and a translation engine Select Translate and a translation engine ](image/e151699c814a1b02.webp)
2. Select `Translate` and a translation engine (Baidu or Google); as you keep selecting different text in the document, the sidebar refreshes the corresponding translation in real time.

   ![When you select text in the document, the Research panel in the sidebar refreshes the translation automatically, in real time. When you select text in the document, the Research panel in the sidebar refreshes the translation automatically, in real time. ](image/ca10c3a0ccd40952.webp)

### 3.2 Using Built-in Research → Dictionary Search

MarginNote supports a variety of authoritative dictionary interfaces, ideal for precise term definitions and example comparison.

![](image/eb8c32cf98e5a281.webp)

Dictionary Search supports several built-in interfaces: Wikipedia (Auto language), Wikipedia (Local language), Oxford, Wikitionary, Baidu, and built-in Research → Custom URL.

### 3.3 Using Built-in Research → Custom URL

> 💡Besides the `Translate` function built into the native Research feature, `Custom URL` lets you connect frequently used translation/dictionary sites to the Research sidebar, so you can look up a word as soon as you select it.

   ![Edit the Custom URL Edit the Custom URL ](image/6f918bb86132196c.webp)
