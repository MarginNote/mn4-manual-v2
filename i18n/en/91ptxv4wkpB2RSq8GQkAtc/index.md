# Auto-Generate Mind Maps ①: One-Tap Excerpting with AI Models

MarginNote's one-tap excerpt model can recognize text, titles, images, image captions, tables, table captions, headers, footers, references, and formulas, so you no longer need to manually draw and adjust selection boxes. `Assistive excerpt` helps you select the elements you need more quickly and accurately.

> 💡 Quick overview of this page's features:
> 
> -  Assistive excerpt settings: automatically recognize elements in the document such as "text, titles, images, image captions, tables, table captions, headers, footers, references, and formulas"
> - Content block assisted recognition: double-tap a thin dashed box to make a single excerpt; or tap the top-right icon to organize with `Merge Cards`, `Become a Child Card`, and more
> - One-tap excerpt (Max feature): batch-excerpt the excerptable elements already shown to quickly generate cards

## 1  Assistive Excerpt Settings

Excerpt Tool Settings

> 💡 Recommended excerpting principle: filter your elements first, then excerpt efficiently.

1. Tap `Excerpt Tool Settings` - enable `Assistive Excerpt`

![Enable Assistive Excerpt Enable Assistive Excerpt ](image/f213e8e2195b571b.webp)

1. In the element toggles, keep only the elements you need.

![Enable assistive excerpt and keep the elements you need Enable assistive excerpt and keep the elements you need ](image/f7a202baa3e221ab.webp)

1. In the document:
   1. Tap once to show the assistive excerpt regions;
   2. Double-tap a region to select it and bring up the `menu bar`;
   3. Double-tap again to excerpt it into a card.

![Tap to select; double-tap to bring up the menu bar; double-tap again to excerpt into a card. Tap to select; double-tap to bring up the menu bar; double-tap again to excerpt into a card. ](image/1fc76dd2f92eacdf.webp)

> 💡 Suggestions for common scenarios
> 
> - Excerpting papers: enable only `Title`, `Text`, `Image`, `Image Caption`, `Table`, `Table Caption`, `Reference`, and `Formula`.
> - Course handouts that mix text and images: enable `Image`, `Image Caption`, and `Text`, do a quick one-tap excerpt, then organize them into child cards.
> - Formula-heavy textbooks: enable `Formula` and `Body Text`, and use single excerpts to keep expressions accurate.

> ⚠️ Note: the assistive excerpt feature increases power consumption and may cause crashes on some low-performance devices, so we recommend enabling it only when you truly need it.

## 2 Content Block Assisted Recognition

> 💡 Beyond quick excerpting, MarginNote 4 supports one-tap recognition of content blocks, so you no longer need to manually box-select images and formulas.

1. After enabling `Assistive Excerpt`, tap a target element, then tap the top-right icon to bring up the `pop-up menu bar`.

![Bring up the pop-up menu bar Bring up the pop-up menu bar ](image/583de560850e9534.webp)

1. Quickly act on the target content.

### 2.1 Merge Cards

Building on content block assisted recognition, tap the top-right icon to select the target content, then drag to merge cards.

![Animated demo: drag to merge cards Animated demo: drag to merge cards ](image/6d281a06a930fa1a.webp)

### 2.2 Become a Child Card

Building on content block assisted recognition, tap the top-right icon to select the target content, then drag to merge cards or set it as a child card.

![Animated demo: drag to merge cards or set as a child card Animated demo: drag to merge cards or set as a child card ](image/0715241bf0403480.webp)

## 3 One-Tap Excerpt (Max Feature)

> 💡 Best for: excerpting short papers and quickly generating cards from every element in an entire document.

Select the elements you want to excerpt, then tap `One-Tap Excerpt` in `Assistive Excerpt`.

![One-tap excerpt to quickly build a card set One-tap excerpt to quickly build a card set ](image/3e2a3d077eec1de6.webp)

> 💡 After excerpting, cards of the same element type are given the same color.

## 4 FAQ

- Can't see the thin dashed boxes?
  - Make sure `Assistive Excerpt` is enabled and that the element toggles aren't all turned off.
- One-tap excerpt grabbed too much content you don't need?
  - Go back and turn off the toggles for the unneeded elements, then use `One-Tap Excerpt` again to select only the useful ones.
  > ⚠️ Note: the assistive excerpt feature increases power consumption and may cause crashes on some low-performance devices, so we recommend enabling it only when you truly need it.
