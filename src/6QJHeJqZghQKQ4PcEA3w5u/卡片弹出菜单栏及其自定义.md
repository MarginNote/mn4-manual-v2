# 3 菜单栏功能详解

## 3.1 基础菜单栏

### 3.1.1 添加新卡片

[添加新卡片](https://www.wolai.com/fZVYVwngNnohGNroR9ji8Y "添加新卡片")

点击`基础`-最左侧“`＋`”符号（如上图所示），可在该卡片的相关层级添加新卡片

- **添加新子卡片**：在下一个层级新建一个空白卡片，排序在该层级的最末尾。
- **添加新同级卡片**：在同一个层级新建一个空白卡片，排序与该卡片紧邻。
- **添加新上级卡片**：在上一个层级新建一个空白卡片，排序在该卡原本的位置。
  > 💡该功能可以缩短“添加同级卡片”→“将同层级卡片移入下层级卡片”→“调整该分支顺序”这一操作链路

### 3.1.2 进行卡片编辑

[卡片编辑器](https://www.wolai.com/rdhVCYTJL3YakoYW4QxtzC "卡片编辑器")

点击`基础`-卡片编辑器图标（如上图所示），可以深入修改卡片内容，具体可参考：[脑图卡片①：新建和编辑卡片](https://www.wolai.com/67hDfmy3SZi4oXmMb2bvcH "脑图卡片①：新建和编辑卡片")

### 3.1.3 改变卡片颜色

[选择颜色](https://www.wolai.com/itZKksYuLFJaUyM7JJ7oDw "选择颜色")

点击`基础`-选择颜色图标（如上图所示），可以修改卡片颜色，也可以修改卡片对应摘录的颜色和高亮方式

### 3.1.4 选择卡片标签

[标签](https://www.wolai.com/cSrqc8HSRgermXHoSJvNTf "标签")

点击基础-标签图标（如上图所示），打开标签界面，可以为卡片添加多个标签，也可以进行`添加`、`删除`、`重命名标签`等操作。

> 💡在**脑图导航栏**和**卡片分组看板**中搜索卡片时，可以点击筛选图标（如下所示），根据**标签和卡片颜色**筛选需要的卡片
>
> [筛选](https://www.wolai.com/gGdFJPvWQDusSqoXe9N7TR "筛选")

### 3.1.5 剪切、复制和粘贴卡片

[剪切](https://www.wolai.com/bpNyzEyiASt8wGBUtwSjAV "剪切")

[复制](https://www.wolai.com/H6nhkfEJUyQXs43YqThzY "复制")

[粘贴](https://www.wolai.com/6UpgK22t2D1kHqnJvtmigH "粘贴")

- 点击剪切图标，出现`卡片被剪切到剪贴板`则剪切成功

> 💡剪切后原卡片不会立即消失，而是等到粘贴到其他为止后再消失，防止忘记剪切过的卡片。

- 点击复制图标，出现`已复制x张卡片的克隆/引用到剪切板`则复制成功。
  > 克隆和引用的区别：**克隆**节点是一个节点卡片的复刻，**引用**节点则是一个节点的镜像。如对卡片 A 进行**克隆**得到卡片 B，修改卡片 A，卡片 B **不会同步**卡片 A的修改；如对卡片 A 进行**引用**得到卡片 B，卡片 B **会同步**卡片 A的修改。详见：[卡片复制：克隆和引用](https://www.wolai.com/pTDpD5xc9XAoB3jSUScm9g "卡片复制：克隆和引用")
- 复制卡片后点击粘贴：
  - 可以轻点脑图空白处进行粘贴，
  - 也可以点击其他卡片，在上弹菜单中选择`粘贴`，这样复制的卡片就成为该卡片的子节点。

### 3.1.6 修改该卡片的分支样式

> 💡只有存在子卡片的卡片，才会在基础上弹菜单显示“`分支`”功能

[分支](https://www.wolai.com/3JmBrm2aQRChW8iwJsDq78 "分支")

点击`基础`-分支图标，出现`分支样式`弹窗，共有10种分支样式，可以根据个人使用习惯和脑图需求选择。

![分支样式弹窗 分支样式弹窗 ](image/IMG_1390_nKpGv52rCU.jpeg "分支样式弹窗 分支样式弹窗 ")

- [**树形**](https://www.wolai.com/6QJHeJqZghQKQ4PcEA3w5u#ski19HMYWvurKMv6VmNww7 "树形")：同级卡片沿竖向分布，最常见的分支样式
- [**直线**](https://www.wolai.com/6QJHeJqZghQKQ4PcEA3w5u#wJgcrCaGaHf8r9XE2JLVJd "直线")：同级卡片沿横向分布，由直线串联，常用于表示时间轴
- [**双向**](https://www.wolai.com/6QJHeJqZghQKQ4PcEA3w5u#ouNzXtjqDVbEBoXhmY4ZJ9 "双向")：同级卡片沿竖向分布，但会均匀分布在上级卡片的两侧，常用于同级卡片过多时
- [**框架**](https://www.wolai.com/6QJHeJqZghQKQ4PcEA3w5u#4Qhodu2JcXxR2PuRtw95Qr "框架")：上级卡片显示为框架，子卡片密集排列在框架中，常用于希望尽可能多的在页面上展示卡片时，例如作为未整理的卡片的集合

[向AI提问](https://www.wolai.com/rcKVh1HiKybKzJrNveYMWa "向AI提问")

详情参考：[卡片链接①|单双向链接：跨越层级的自由关联](https://www.wolai.com/mGz8BQh6r1ad6wv1jqDo1G "卡片链接①|单双向链接：跨越层级的自由关联")

### 3.1.9 焦点

点击`焦点`，该卡片以及其子卡片会进入一个空白脑图中，暂时隐藏其他无关内容，专注梳理目标分支。

详情参考：[脑图焦点](https://www.wolai.com/5nntHuyzfBSdVrEo4YYhxC "脑图焦点")

### 3.1.10 删除

[删除](https://www.wolai.com/6WFcR1Djfow4F7YWkkPqaK "删除")

点击`删除图标`（如上图所示），选择的卡片会被删除。如果该卡片有子卡片，则会弹出删除卡片/删除卡片树选项

- 选择`删除卡片`：只删除该卡片，下方层级的卡片会自动上升层级
- 选择`删除卡片树`：该卡片及该卡片下方所有层级的卡片一并删除。

![](image/30400a9be0cbae2de9d22ec35dcf2195_grDLpi1RbR.png.mark.png)

| 序号 | 名称         | 功能                                                                                                                                                                            | 备注             |
| -- | ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------- |
| 1  | 添加复习       | 将选中的卡片从加入复习卡片组                                                                                                                                                                | 仅限未添加到复习卡片组的卡片 |
| 2  | H1         | 点击后，卡片正文第一条摘录会转换为标题  > 💡与图中未标出的区别在于，在有标题的情况下点击H1，标题会转换为卡片正文。                                                                                                                 |                |
| 3  | 基于分支生成拆书模板 | 点击\`基于分支生成拆书模板，\`可以由ai分析思维脑图结构，并生成基于个性化脑图的拆书模板，该模板可应用于文档界面\[目录]\(<https://www.wolai.com/jaxFYszwK9eiofzRPxzA1J> "目录")中的ai拆书                                                   |                |
| 4  | 在浏览器中搜索    | 点击该图标，在脑图侧边栏会打开\[研究]\(<https://www.wolai.com/7AXUEU5B5UUuEGW1m7oZ9z> "研究")，并自动搜索卡片内容，详情参考：\[检索③：文档与研究浏览器联动]\(<https://www.wolai.com/nfvG3Hah2KmZUXGaD4UJh1> "检索③：文档与研究浏览器联动") |                |
| 5  | OCR校正      | 点击\`OCR校正\`，该卡片的\*\*摘录部分\*\*会被重新进行ocr，并更新卡片中文字                                                                                                                                | 仅限有摘录的卡片       |
| 6  | 文本         | 点击\`文本\`，卡片中的\*\*图片\*\*会被校正为文本格式                                                                                                                                              | 仅限形式为图片的摘录     |
| 7  | 导出         | 可将卡片导出为pdf或长图格式                                                                                                                                                               |                |
