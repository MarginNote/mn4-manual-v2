## 2 如何“导出到Anki”？

- 方法一：从首页的“复习卡片组”导出

  复习卡片组 → 选择卡片组 → 导出 → 导出到Anki (apkg)

  ![](image/20240505213900-convert_aZ2IhlT7Qa.gif)
- 方法二：从学习集的“复习”导出

  复习 → … (更多) → 导出到Anki (apkg)

  ![](image/20240505214258-convert_WbEY18qNIJ.gif)
- 方法三：从学习集的“更多”导出

  &#x20;… (更多) → 导出学习集 → 导出到Anki (apkg)

  ![](image/20240505214520-convert_v810lzBtjk.gif)

## 4 导出：使用“自定义模板”

![Basic 3in1 自定义模板 Basic 3in1 自定义模板 ](image/image_NzTKCNGNP3.png "Basic 3in1 自定义模板 Basic 3in1 自定义模板 ")

- 笔记卡片数据的具体含义
  | MarginNote卡片数据     | 对应数据          | 含义                               |
  | ------------------ | ------------- | -------------------------------- |
  | MarginNote卡片正面（问题） | #CardQuestion | 卡片的正面                            |
  | MarginNote卡片背面（笔记） | #NoteWhole    | 卡片的背面（即**卡片上的所有笔记**）             |
  | 卡片的标题部分            | #NoteTitle    | 卡片的标题                            |
  | 卡片的摘录部分            | #NoteExcerpt  | 卡片的摘录 （从文档中摘录的**有源笔记**）          |
  | 卡片的评论部分            | #NoteComment  | 卡片的评论（包括标签、语音评论）                 |
  | 备注 （用于搜索）          | #Remark       | 卡片中的所有文本内容（包括OCR识别结果），以便在Anki中检索 |
  | MarginNote回链       | #MNLink       | 该链接可跳转至卡片在MarginNote中的脑图位置       |
  | 子节点链接              | #MapLink      | 该卡片的子节点，可在Anki中生成图片预览            |

### 4.1 如何使用自定义模板？

- 使用自定义模板的具体步骤（以“Basic 3in1 模板”为例）
  - 先将“Basic 3in1 模板”保存到“本地文件”
  - 导出时切换到“自定义模板” → 点击“选择模板 ( \*.apkg)” → 选择保存到“本地文件”的自定义模板

    ![](image/20240505223434-convert_wUq8pIWhLQ.gif)
  - 从字段列表中选择需要的MarginNote卡片数据填充到Anki字段

    ![](image/20240505224725-convert_QenqjWdwFp.gif)
  - 选择Anki → 点击右上角“导入”即可

    ![](image/20240505225224-convert_MSznerTk6-.gif)

> 💡如果想要在自定义模板中使用到“导出到MarginNote的链接”、“导出子链接为图片”、“导出语音评论”这些笔记数据，请先在默认模板阶段勾选对应导出项，方能导出！

### 7.1 通过快捷指令导出到Anki

- 1、安装快捷指令“Export to Anki”快捷指令

  [Shortcuts  https://www.icloud.com/shortcuts/b6c5ae3f0a5241019f918a121b318824](https://www.icloud.com/shortcuts/b6c5ae3f0a5241019f918a121b318824 "Shortcuts  https://www.icloud.com/shortcuts/b6c5ae3f0a5241019f918a121b318824")
- 2、根据实际需求，按照AnkiMobile的URL方案，适当调整快捷指令
  - AnkiMobile的URL方案
    - 添加笔记的固定开头：
      > anki://x-callback-url/addnote?
    - 以下参数可以根据需要进行选择：
      | 参数        | 含义       |
      | --------- | -------- |
      | profile   | 用户名      |
      | type      | 笔记模板名    |
      | deck      | 牌组名      |
      | fld       | 字段输入前缀   |
      | tags      | 标签       |
      | dupes=1   | 允许相同添加笔记 |
      | x-success | 笔记添加后返回  |
    - 更多详细参数可见：

      [URL Schemes - AnkiMobile Manual  https://docs.ankimobile.net/url-schemes.html](https://docs.ankimobile.net/url-schemes.html "URL Schemes - AnkiMobile Manual  https://docs.ankimobile.net/url-schemes.html")
  - 调整快捷指令

    修改或添加笔记模板名、牌组名、字段名、标签等。

    ![修改快捷指令参数 修改快捷指令参数 ](image/IMG_372_mpg0xhu_ro.jpeg "修改快捷指令参数 修改快捷指令参数 ")
- 3、运行“Export to Anki”快捷指令
  - 选中卡片 → 打开“卡片编辑器” → … (更多) → 运行“Export to Anki”快捷指令；
  - 先选择问题，再选择答案，可多选；
  - 点击“完成”，即可导出到Anki中。
    ![运行 “Export to Anki” 快捷指令 运行 “Export to Anki” 快捷指令 ](image/20240503174728-convert_81vU9wjEqV.gif "运行 “Export to Anki” 快捷指令 运行 “Export to Anki” 快捷指令 ")
